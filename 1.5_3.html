Атрибути

Опис атрибутів які допустимі дя тегу (зробити таблицею)

type встановлює вид маркера списку
accesskey дозволяє отримати доступ до елемента за допомогою комбінації клавіш
class визначає ім'я класу
class визначає ім'я класу яке дозволяє зв'язати елемент зі стилевим оформленням
contenteditable Сповіщає, що елемент доступний для редагування користувачем
contextmenu встановлює контекстне меню для елементу
data-* дозволяє створювати свої атрибути
hidden приховує вміст елементу від перегляду
id вказує ім'я стилевого ідентифікатору
